About the page

Routines are a way of managing the classroom. They are a set of expectations that save time and ensure smooth-functioning classroom. We are conceptualizing this software as a solution to the institutions so that every teacher should be provided a balanced mix of tasks, their work load is defined and the overall system's function should be uniform. In the Existing System, everything is done manually, which means generating/maintaining reports and that leads to time consuming as well as increases burden. The Proposed System have overcome this problem by making the operations computerized, which were done manually.
